# random-circle-mover

Create a web app that contains a random number of circles 10 pixels in diameter, arranged in random groups of 1 to 5 circles. When a user mouses over a circle it moves to a random location 200 pixels away.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/random-circle-mover.git
cd random-circle-mover
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
